Bantuan Interaktif untuk Pengaturan Hesk

 

» Pengaturan Umum

Judul Website

Judul pendek dari website Anda, misalnya "Website indah saya". Ini akan digunakan sebagai judul tautan yang ditampilkan di sudut kiri atas antarmuka pelanggan untuk memungkinkan mereka kembali ke website Anda.

URL Website

URL website Anda, misalnya "http://www.example.com". Pastikan ini adalah URL yang valid dan Anda memulainya dengan http:// atau https:// . Ini akan digunakan sebagai URL tautan yang ditampilkan di sudut kiri atas antarmuka pelanggan dan memungkinkan mereka kembali ke website Anda.

Email Webmaster

Alamat email webmaster default. Pastikan Anda memasukkan alamat email yang valid, bisa sama dengan email Dukungan.

 

Tutup jendela

 

"From:" email

Alamat email "From:" Anda. Ini adalah alamat email yang akan ditampilkan sebagai pengirim email (header From:) di semua email yang dikirim oleh HESK. Biasanya ini adalah email yang tidak Anda baca dan tidak Anda balas. Pastikan Anda memasukkan alamat email yang valid, bisa sama dengan email Dukungan tetapi disarankan untuk menggunakan yang berbeda.

"From:" nama

Ini adalah nama yang akan ditampilkan sebagai pengirim email (From:) di semua email yang dikirim oleh HESK. Jika tidak ada nama yang disetel, header From: pada email hanya akan menampilkan alamat email From: Anda.

Tema situs

Ini adalah tema yang akan digunakan di semua halaman HESK yang menghadap pelanggan. Semua tema yang terpasang dengan benar akan ditampilkan di kotak drop-down. Tombol "Uji folder tema" akan membantu Anda memecahkan masalah tema yang tidak berfungsi.

Admin CSS

Di sini Anda dapat menentukan URL file CSS kustom yang akan disertakan dalam panel admin.

Gunakan file CSS kustom Anda untuk mengganti gaya panel admin default HESK atau menambahkan aturan CSS tambahan tanpa memodifikasi file sumber.

Admin JS

Di sini Anda dapat menentukan URL file Javascript kustom yang akan disertakan dalam panel admin.

Gunakan file Javascript kustom Anda untuk mengganti Javascript panel admin default HESK atau menambahkan Javascript tambahan tanpa memodifikasi file sumber.

» Pengaturan Bahasa

Bahasa Default

Jika Anda telah menginstal beberapa bahasa untuk Hesk, ini akan menjadi bahasa default. Semua bahasa yang terpasang dengan benar ditampilkan di kotak drop-down. Silakan merujuk ke file readme.html untuk petunjuk tentang cara menginstal lebih banyak bahasa. Tautan "Uji folder bahasa" akan membantu Anda memecahkan masalah instalasi bahasa yang tidak berfungsi.

Jika beberapa bahasa diaktifkan, Anda dapat mengatur bahasa pilihan Anda di halaman "Profil".

Beberapa bahasa

Mengaktifkan pengaturan ini akan memungkinkan pengguna dan staf Anda memilih bahasa pilihan mereka dari daftar bahasa yang diinstal. Jangan aktifkan fitur ini kecuali Anda memberikan dukungan dalam semua bahasa yang diinstal!

» Pengaturan Database

Host Database

Server host database, "localhost" berfungsi untuk sebagian besar pengguna. JANGAN mengubah ini kecuali Anda tahu host MySQL Anda. Tanyakan kepada perusahaan hosting Anda jika tidak yakin atau jika Anda mengalami masalah menghubungkan ke database MySQL.

Nama Database

Nama database MySQL Anda. Anda harus membuatnya di panel kontrol hosting web Anda atau meminta informasi ini dari perusahaan hosting Anda.

Nama Pengguna Database

Nama pengguna yang Anda gunakan untuk menghubungkan ke database MySQL Anda. Anda harus membuat pengguna MySQL di panel kontrol hosting web Anda atau meminta informasi ini dari perusahaan hosting Anda.

 

Tutup jendela

 

Kata Sandi Database

Kata sandi yang Anda gunakan untuk menghubungkan ke database MySQL. Anda harus membuat pengguna MySQL di panel kontrol hosting web Anda atau meminta informasi ini dari perusahaan hosting Anda.

Prefiks tabel

Prefiks yang akan ditambahkan ke semua nama tabel. Jika diatur ke hesk_ tabel "users" sebenarnya akan dinamai "hesk_users" di database Anda. Ini berguna jika Anda ingin menjalankan beberapa instalasi Hesk menggunakan satu database MySQL, cukup beri setiap instalasi nama prefiks tabel yang unik (hesk1_, hesk2_, hesk3_, ...)
